# 基础篇

## Linux版本

### 内核不同分类

- xx.yy.zz

	- xx主版本
	- yy次版本
	- zz末版本

- 小于2.6版本

	- 次版本奇数为开发版
	- 次版本偶数为稳定版

- 大于2.6版本

	- Longterm长期支持版本
	- stable稳定版本
	- mainline主线开发版本

### 常见Linux版本

- Red Hat
- Ubuntu
- CentOS
- Debian
- Fedora
